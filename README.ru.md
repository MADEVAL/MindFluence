# MindFluence

[ [English](README.md) | **Русский** ]

> **Убеждение, построенное на психологии, а не на манипуляции.**

AI-навык (промпт), превращающий 20 когнитивных искажений в маркетинговый копирайтинг с высокой конверсией - для любого языка, любой платформы, любой аудитории.

---

## Что это?

**MindFluence** - это системный промпт (навык) для LLM - GPT, Claude, Gemini, DeepSeek или любой другой способной модели. Он превращает AI в маркетингового стратега, который понимает, *почему* люди кликают, читают и покупают, а не просто *что* писать.

В основе - десятилетия исследований в области поведенческой экономики и эволюционной психологии: две системы Канемана, принципы Чалдини, психологические ловушки Мангера, когнитивный диссонанс Фестингера - сведённые в практические инструкции по копирайтингу.

## Как это работает

1. **Загрузите `SKILL.md`** как системный промпт в любую LLM.
2. **Дайте задание:** «Напиши пост в LinkedIn о...» / «Создай лендинг для...» / «Проведи аудит объявления».
3. **Получите результат** с каждым искажением под меткой, каждым решением с объяснением, каждым словом с психологическим обоснованием.

**Четыре режима:**
- **Quick Mode** (по умолчанию) - минимум ввода, немедленный результат. AI автоматически выбирает лучшие искажения.
- **Deep Mode** - AI задаёт уточняющие вопросы об аудитории, платформе, тоне и желаемом действии перед генерацией.
- **Audit Mode** - анализирует существующий текст на наличие когнитивных искажений и анти-паттернов. Не создаёт - препарирует.
- **Optimize Mode** - итеративно улучшает неэффективный копирайтинг на основе метрик (CTR, конверсия, open rate). Сопоставляет этапы воронки с конкретными искажениями, заменяет провальные искажения, генерирует A/B-варианты.

## Переключатель стиля (Tone-of-Voice)

Каждый результат помечается одним из 5 стилей (или гибридом):

| Стиль | Голос | Лучше всего для |
|--------|-------|------------------|
| `bold-sell` | Агрессивный, срочный, ударный | Флеш-распродажи, DTC, запуски |
| `expert-calm` | Интеллектуальный, размеренный, с данными | B2B, SaaS, консалтинг |
| `rebel-edgy` | Дерзкий, контркультурный | Молодёжные бренды, челленджеры, создатели |
| `warm-human` | Эмпатичный, сюжетный | Здоровье, коучинг, сообщества |
| `luxe-minimal` | Отточенный, лаконичный, высокий статус | Премиум, люкс, дизайн |

## Комбинации искажений

Некоторые искажения умножают эффект при комбинировании:

| Комбо | Искажения | Для чего |
|--------|-----------|----------|
| **Спираль доверия** | Авторитет → Соцдоказательство → Подтверждение → Владение | Лендинги, продающие страницы |
| **Двигатель срочности** | Страх → Неприятие потери → Соцдоказательство → Дефицит | Флеш-распродажи, запуски |
| **Петля лояльности** | Подтверждение → Своя группа → Невозвратные затраты → Статус-кво | Удержание, допродажи |
| **Цепочка конверсии** | Доступность → Фрейминг → Якорение → Соцдоказательство → Снятие риска | Реклама, переход от бесплатного к платному |

## Матрица решений

Навык не угадывает, какие искажения использовать. `decision-matrix.md` предоставляет справочную таблицу:

- **Температура аудитории** (холодная / тёплая / горячая / ушедшая / скептичная) → приоритет искажений
- **Тип продукта** (low-B2C, high-B2C, B2B SaaS, инфопродукты, здоровье, сообщество) → доминирующие искажения
- **Ограничения платформы** (Twitter, LinkedIn, Instagram, email, лендинг, вебинар) → разрешённые и запрещённые искажения
- **Пошаговый алгоритм:** аудитория → продукт → платформа → комбинировать → проверить
- **Быстрая справочная таблица** для 18 типовых задач с предварительно рассчитанными стеками искажений

## Антипаттерны

`anti-patterns.md` фиксирует 12 самых частых способов, которыми AI ошибается в копирайтинге на когнитивных искажениях - каждый с провальным выводом, психологической причиной провала, конкретным исправлением и правилом:

| # | Антипаттерн | Пример провала |
|---|-------------|-----------------|
| 1 | Размытое соцдоказательство | «Тысячи довольных клиентов» |
| 2 | Страх без выхода | Угроза без немедленного решения |
| 3 | Фрейминг без якоря | «Это революция» - без точки отсчёта |
| 4 | Авторитет без доказательств | «Исследования показывают...» - без источника, без года |
| 5 | Транзакционная взаимность | «Бесплатная книга! Теперь купи.» - подарок и предложение в одном дыхании |
| 6 | Фальшивый дефицит | Таймеры обратного отсчёта, сбрасывающиеся при перезагрузке |
| 7 | Преждевременные невозвратные затраты | «Вы уже столько прошли» - в первом же абзаце |
| 8 | Пустая «своя группа» | «Единомышленники» - без общей идентичности |
| 9 | Ореол не того источника | «О нас писали в Forbes» - аудитория не уважает Forbes |
| 10 | Оскорбительное подтверждение | «Большинство людей ошибается» - вызывает защитную реакцию |
| 11 | Абстрактная доступность | «Представьте, что вы боретесь» - без конкретных сенсорных деталей |
| 12 | Обвинительный диссонанс | «Вы говорите X, но делаете Y» - обвиняет читателя, а не систему |

Включает **чеклист аудита** - проверьте любой вывод за 30 секунд перед отправкой.

## Этические границы

**Всегда:**
- Используйте искажения, чтобы привлечь внимание к действительно ценным продуктам
- Будьте прозрачны о том, что ваш продукт делает, а что нет
- Используйте фактические, проверяемые социальные доказательства

**Никогда:**
- Не создавайте ложный дефицит
- Не фабрикуйте отзывы или обзоры
- Не эксплуатируйте страх для продажи решений, которые не решают проблему страха
- Не укрепляйте вредные убеждения
- Не нацеливайтесь на уязвимые группы населения

**Грань:** Выбрал бы клиент ваш продукт, обладая полной информацией и временем на размышление? *Да* = убеждение. *Нет* = манипуляция.

## Быстрый старт

### Быстрый старт (любая LLM)

1. Скопируйте содержимое `SKILL.md`
2. Вставьте как системный промпт / пользовательские инструкции
3. Напишите: *«Создай пост на тему [тема] для [аудитория] на [платформа]»*

### С OpenCode (нативно)

Поместите папку `mindfluence/` в `.opencode/skills/` (проект) или `~/.config/opencode/skills/` (глобально). OpenCode автоматически обнаружит `SKILL.md` через его frontmatter - дополнительная настройка не требуется. Навык активируется при запросе маркетинговых текстов, рекламы, лендингов, писем или контента для запусков.

### С Claude Code

Скопируйте `SKILL.md` в ваш проект как `CLAUDE.md` или поместите всю папку `mindfluence/` в `~/.claude/skills/`.

### Как Custom GPT / Проект Claude

Загрузите `SKILL.md` как системный промпт. Добавьте `decision-matrix.md`, `anti-patterns.md` и файлы из `scenarios/` и `examples/` в раздел знаний для полного контекста.

### С любым другим AI-агентом (Cursor, Windsurf, Copilot, Cline, Aider и др.)

Скопируйте содержимое `SKILL.md` в файл инструкций этого агента (`.cursorrules`, `.windsurfrules`, `.github/copilot-instructions.md`, `.clinerules`, `CONVENTIONS.md` и т.д.). Содержимое промпта не зависит от платформы - меняется только имя файла.

## Требования

- Любая способная LLM с полем системного промпта / пользовательских инструкций (GPT, Claude, Gemini, DeepSeek, open-source модели)
- Для полной функциональности навыка (матрица решений, антипаттерны, сценарии, примеры): система навыков, загружающая файлы из папки (OpenCode, Claude Code). Для других агентов скопируйте `SKILL.md` как отдельный промпт - он содержит полную основную методологию.
- Без API-ключей, без инструментов, без зависимостей - чистый промпт-инжиниринг
- Работает на любом языке; адаптирует культурные отсылки под целевой рынок

## Почему «mindfluence»?

**mind** (мозг - здесь живут когнитивные искажения) + **influence** (то, что убеждение на самом деле делает - формирует решения без принуждения).
